# James Michael Walter - Actor Portfolio Website

Professional actor portfolio website for James Michael Walter, based in Los Angeles.

## Features

- Dark, professional aesthetic
- Responsive design (mobile-friendly)
- Sections: Hero, About, Demo Reel, Resume, Gallery, Contact
- Smooth scroll navigation
- Fade-in animations
- SEO-optimized meta tags

## Structure

```
jamesmichaelwalter/
├── index.html      # Main HTML file
├── style.css       # All styling
├── script.js       # Mobile menu & animations
├── README.md       # This file
└── assets/         # Images (to be added)
    ├── headshots/
    └── gallery/
```

## Customization Needed

1. **Headshots**: Replace placeholder divs with actual images
2. **Demo Reel**: Embed your Vimeo/YouTube reel
3. **Resume**: Update with your actual credits and training
4. **About**: Personalize the bio text
5. **Contact**: Update email and social links
6. **Actors Access/IMDb**: Add profile links

## Deployment

This is a static site - can be deployed to:
- Netlify (recommended)
- Vercel
- GitHub Pages
- Any static hosting
